# GitHub Fork Detachment Request

**Repository:** https://github.com/irfndi/goflux  
**Original fork source:** https://github.com/sdcoffey/techan

## Reason for detachment

This repository was originally forked from `sdcoffey/techan`, but it has since evolved into a completely separate project with:

- Different architecture (modular packages, interface-based design)
- 35+ additional technical indicators
- New strategy engine, backtesting framework, and performance metrics
- Cloud-native features (telemetry, D1 database integration)
- Different API surface and package structure
- Active independent development with no intention to merge upstream

The codebase now bears little resemblance to the original fork.

## How to submit

1. Go to https://support.github.com/contact
2. Select "Repository management" → "Detach a fork"
3. Include the repository URL: `https://github.com/irfndi/goflux`
4. Explain that the project has diverged significantly and is now independent

## After detachment

- The repository will appear as a standalone project
- Stars, issues, and PRs will be preserved
- The project will no longer show "forked from sdcoffey/techan"
